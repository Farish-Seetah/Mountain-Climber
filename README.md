README
How to Run the Game
Download or extract the project files.
Open the project folder.
Double-click on index.html to launch the game in your browser.
Controls
Key	Action
W	Move Forward
A	Move Left
S	Move Backward
D	Move Right


Requirements
A modern web browser such as:
Google Chrome
Mozilla Firefox
Microsoft Edge
Notes
Make sure all project files stay in the same folder as index.html.
If textures or assets are missing, ensure the folders were extracted correctly.
Best experienced in fullscreen mode.
